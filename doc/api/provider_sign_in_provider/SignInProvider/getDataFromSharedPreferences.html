Future<void> getDataFromSharedPreferences()

Get user data from SharedPreferences This method retrieves the user's data from SharedPreferences. It uses the SharedPreferences instance to get the user's information. If the data is retrieved successfully, it prints a message to the console. If there is an error retrieving the data, it prints an error message to the console. This method is called when the app is opened to check if the user is already signed in.

Implementation

Future<void> getDataFromSharedPreferences() async {
  final SharedPreferences s = await SharedPreferences.getInstance();
  _name = s.getString('name');
  _email = s.getString('email');
  _imageUrl = s.getString('image_url');
  _uid = s.getString('uid');
  _provider = s.getString('provider');
  notifyListeners();
  print("User data retrieved from SharedPreferences: $_name, $_email, $_provider");
}
